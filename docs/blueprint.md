# **App Name**: Tapi Tapi Rebrand

## Core Features:

- Homepage Design: Visually appealing homepage with welcome message and clear brand introduction.
- Navigation Menu: Intuitive navigation for easy website exploration.
- About Us Page: Informative 'About Us' section detailing Tapi Tapi's history, mission, and values.
- Contact Us Page: Dedicated 'Contact Us' page with fictional contact details for inquiries.
- Call-to-Action Button: Prominent CTA button encouraging user engagement.

## Style Guidelines:

- Primary color: Vibrant Orange (#FF8C00) to represent the warmth and energy of African flavors.
- Background color: Light Beige (#F5F5DC) for a natural, creamy backdrop.
- Accent color: Muted Green (#90EE90) to complement the orange and provide a fresh, organic feel.
- Body and headline font: 'PT Sans', a humanist sans-serif font combining a modern look and a little warmth, for both headlines and body text. Note: currently only Google Fonts are supported.
- Clean and intuitive layout with clear sections for easy navigation.
- Custom icons inspired by African art and ice cream flavors.
